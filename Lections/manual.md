# Материалы по Python

### Арифмитические операции

Умножение: `*`

`a * b`

Деление: `/`

`a / b`

Возведение в степень: `**`

`a ** b`

Остаток от деления: `%`

`a % b`

Целочисленное деление: `//`

`a // b`

### Списки

Копирование списка (срез): `a = b[:]`

``` b = [1, 2, 3]
    a = b[:]
    b.append[4]

    Console:    a = [1, 2, 3]
                b = [1, 2, 3, 4]
```

Ссылка на список: `a = b`

``` b = [1, 2, 3]
    a = b
    b.append[4]

    Console:    a = [1, 2, 3, 4]
                b = [1, 2, 3, 4]
```
